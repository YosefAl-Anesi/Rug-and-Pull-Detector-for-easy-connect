# Rug and Pull Detector

A tool to detect potential rug pulls in crypto using EasyConnect integrations with Make.com.


## How it Works
1. **EasyConnect** monitors transactions on the Qubic blockchain.
2. Alerts are sent for:
   - Insider Dump – Team Wallet
   - Liquidity Alert – Add/Remove From Ask Order
   - Vesting Alerts
3. **Make.com** receives these alerts via webhook and posts them to Discord in real-time.
4. Users can see potential risks instantly and take action.
